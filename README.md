# xyzapi

Small Pacakage npm :)

# instalation
```shell
npm install xyzapi
```
Or via yarn
```shell
yarn add xyzapi
```

# Usage

Example!

### Islami

```List Surah```
```javascript
const xyz = require('xyzapi')

const apikey = 'APIKEY_KAMU' //apikey bisa kalian dapatkan gratis dengan cara register di https://api.darkcoder.xyz

xyz.islami.listsurah(apikey)
.then(result => {
    console.log(result)
})
.catch(error => {
    console.log(error)
})
```
